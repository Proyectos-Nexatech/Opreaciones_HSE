# Implementación de Autenticación Completada

Se ha implementado el sistema de autenticación completo en la aplicación.

## Características Implementadas
1. **Protección de Rutas**: Acceder a cualquier página sin sesión redirige a `/login`.
2. **Registro**: Nueva página `/register` para crear cuentas.
3. **Login**: Página `/login` funcional.
4. **Header**: Muestra el usuario actual y botón de Cerrar Sesión.

## Pasos para probar
1. Refresca la página.
2. Si no estás logueado, serás redirigido al Login.
3. Crea una cuenta o inicia sesión.
